# Photo-Site

## To open the project

### `First`
Open the project directory in your IDE.

### `Second`
Open 2 terminals:

In the first type:
  1. "cd server".
  2. "npm install".
  2. After the previous operation is done - "npm start".

In the second type:
  1. "cd client".
  2. "npm install".
  2. After the previous operation is done - "npm start".
